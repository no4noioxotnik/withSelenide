    Перевод "hack this" на русский

    Искать hack this в:
    взломать эту
    You ready to hack this system?
    Готова взломать эту систему?
    Maybe I can't hack this.
    Может, мне это не провернуть.
    I don't think I can hack this.
    Сомневаюсь, что я это всё осилю.
    I can't hack this any more.
    Я не могу больше это терпеть.
    I need you to hack this X-ray machine.
    Мне надо, что бы ты "хакнул" сканер.
    If we can hack this mainframe, we'll have the identity of every agent in Jones's network.
    Если мы сможем взломать этот компьютер, у нас будут личности каждого агента в сети Джонса.
    Did you actually hack this hotel from CTOC?
    Ты что, взломал этот отель?
    I can't hack this for the rest of my life, Rich, all right?
    Я не богу разбить всю свою оставшуюся жизнь, Рич, ясно?
    Right, hack this!
    Хорошо, взломай это!
    I'm going to hack this stuff.
    А я пока займусь этой штукой
    Come on, you can hack this.
    Давай, ты справишься.
